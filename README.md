# Basic Django Website

This is a minimal Django web application created to demonstrate the basic features of Django, including:

- Project and app setup
- URL routing
- Views and templates
- Running a development server

## Pages

1. Home – Main landing page
2. About – Simple information page
3. Contact – Basic contact page

## Features

- Fully functional Django project structure
- Easy to extend for additional pages or apps
- Suitable for beginners learning Django
- Can be deployed locally or on a web server
